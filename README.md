# Battery Formation QC Analyzer

Python mini project for analyzing battery cell formation QC data using Pandas and NumPy.

# Features
1. Handles missing OCV values
2. Cleans invalid data
3. PASS/FAIL classification
4. Failure rate calculation
5. QC report generation

## Technologies Used
 1. Python Language
 2. Pandas liabrary
 3. NumPy liabrary

## Run
bash
pip install pandas numpy
python Formation_QC_Analyzer.py
